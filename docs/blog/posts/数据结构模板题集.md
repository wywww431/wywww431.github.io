## 1. 链表 (Linked List) — 反转链表

> PPT: DS02_Ch03_List

### 题目

给定单链表头节点 `head`，将其反转并返回新头节点。

### 模板代码

```cpp
typedef struct ListNode {
    int val;
    struct ListNode *next;
} ListNode;

ListNode* reverseList(ListNode* head) {
    ListNode *prev = NULL;
    ListNode *curr = head;
    while (curr) {
        ListNode *next = curr->next;  // 先记录下一个
        curr->next = prev;            // 反转指针
        prev = curr;                  // prev 前进
        curr = next;                  // curr 前进
    }
    return prev;
}
```

| 操作 | 复杂度 |
|------|--------|
| 时间 | $O(n)$ |
| 空间 | $O(1)$ |

> 🔑 **关键**：三个指针 `prev`, `curr`, `next` 交替前进，画图理解。

## 2. 栈 (Stack) — 出栈序列合法性判断

> PPT: DS03_Ch03_Stack and Queue

### 题目

给定入栈顺序 $1, 2, \dots, N$ 和一个出栈序列，判断是否合法。栈容量为 $M$。

### 模板代码

```cpp
#include <stdio.h>
#define MAXN 1000

int isValid(int M, int N, int seq[]) {
    int stack[MAXN], top = -1;
    int cur = 1;  // 下一个要入栈的数

    for (int i = 0; i < N; i++) {
        int target = seq[i];
        // 栈顶就是目标，直接出栈
        if (top >= 0 && stack[top] == target) {
            top--;
            continue;
        }
        // 否则一直入栈直到目标出现（或栈满）
        while (top == -1 || stack[top] != target) {
            if (top == M - 1) return 0;  // 栈满，不可能
            stack[++top] = cur++;
        }
        top--;  // 弹出 target
    }
    return 1;
}
```

| 操作 | 复杂度 |
|------|--------|
| 时间 | $O(N)$ |
| 空间 | $O(M)$ |

> 🔑 **关键**：用一个变量 `cur` 追踪**下一个入栈的数**，模拟真实的 push/pop 过程。

## 3. 队列 (Queue) — 约瑟夫环

> PPT: DS03_Ch03_Stack and Queue

### 题目

$N$ 个人围成一圈，从第 1 个人开始报数，报到 $M$ 的人出列，下一个人重新从 1 报数。输出出列顺序。

### 模板代码

```cpp
#include <stdio.h>
#define MAXN 1000

void josephus(int N, int M) {
    int q[MAXN], front = 0, rear = 0;

    // 1 到 N 入队
    for (int i = 1; i <= N; i++)
        q[rear++] = i;

    while (front < rear) {
        // 前 M-1 个人出队再入队
        for (int i = 1; i < M; i++) {
            q[rear++] = q[front++];
        }
        // 第 M 个人出列
        printf("%d ", q[front++]);
    }
}
```

| 操作 | 复杂度 |
|------|--------|
| 时间 | $O(N \cdot M)$ |
| 空间 | $O(N)$ |

> 🔑 **关键**：用队列模拟循环——不是目标就出队再入队，是目标就只出队不入队。

## 4. 二叉树 (Binary Tree) — 三种遍历

> PPT: DS04_Ch04_Binary Trees

### 题目

给定二叉树，实现前序、中序、后序遍历（递归版 & 非递归版）。

### 模板代码（递归版）

```cpp
typedef struct TreeNode {
    int val;
    struct TreeNode *left, *right;
} TreeNode;

void preorder(TreeNode *root) {
    if (!root) return;
    printf("%d ", root->val);
    preorder(root->left);
    preorder(root->right);
}

void inorder(TreeNode *root) {
    if (!root) return;
    inorder(root->left);
    printf("%d ", root->val);
    inorder(root->right);
}

void postorder(TreeNode *root) {
    if (!root) return;
    postorder(root->left);
    postorder(root->right);
    printf("%d ", root->val);
}
```

### 模板代码（非递归中序）

```cpp
void inorder_iter(TreeNode *root) {
    TreeNode *stack[1000];
    int top = -1;
    TreeNode *cur = root;

    while (cur || top >= 0) {
        // 一路向左，沿途入栈
        while (cur) {
            stack[++top] = cur;
            cur = cur->left;
        }
        // 弹出并访问
        cur = stack[top--];
        printf("%d ", cur->val);
        // 转向右子树
        cur = cur->right;
    }
}
```

| 遍历 | 递归 | 非递归关键 |
|------|------|-----------|
| 前序 | `根-左-右` | 栈 + 先右后左入栈 |
| 中序 | `左-根-右` | 栈 + 一路向左 |
| 后序 | `左-右-根` | 双栈 或 prev指针标记 |
| 层序 | — | 队列 BFS |

> 🔑 **关键**：递归三行代码的顺序就是遍历顺序，非递归用栈模拟递归调用栈。

## 5. 二叉搜索树 (BST) — 插入与查找

> PPT: DS05_Ch04_Search Tree

### 题目

实现 BST 的插入、查找、找最小值操作。

### 模板代码

```cpp
typedef struct TreeNode {
    int val;
    struct TreeNode *left, *right;
} TreeNode;

// 插入（递归）
TreeNode* insert(TreeNode *root, int x) {
    if (!root) {
        root = (TreeNode*)malloc(sizeof(TreeNode));
        root->val = x;
        root->left = root->right = NULL;
        return root;
    }
    if (x < root->val)
        root->left = insert(root->left, x);
    else if (x > root->val)
        root->right = insert(root->right, x);
    return root;
}

// 查找
TreeNode* find(TreeNode *root, int x) {
    if (!root) return NULL;
    if (x < root->val) return find(root->left, x);
    if (x > root->val) return find(root->right, x);
    return root;
}

// 找最小值
TreeNode* findMin(TreeNode *root) {
    if (!root) return NULL;
    while (root->left) root = root->left;
    return root;
}

// 删除（重点！）
TreeNode* deleteNode(TreeNode *root, int x) {
    if (!root) return NULL;
    if (x < root->val)
        root->left = deleteNode(root->left, x);
    else if (x > root->val)
        root->right = deleteNode(root->right, x);
    else {
        // 找到要删的节点
        if (!root->left) {       // 只有右孩子 / 叶子
            TreeNode *tmp = root->right;
            free(root);
            return tmp;
        }
        if (!root->right) {      // 只有左孩子
            TreeNode *tmp = root->left;
            free(root);
            return tmp;
        }
        // 有两个孩子：找右子树最小值替换
        TreeNode *min = findMin(root->right);
        root->val = min->val;
        root->right = deleteNode(root->right, min->val);
    }
    return root;
}
```

| 操作 | 平均 | 最坏（退化成链表） |
|------|------|--------------------|
| 查找/插入/删除 | $O(\log n)$ | $O(n)$ |

> 🔑 **删除两个孩子的节点**：用右子树的最小值（或左子树的最大值）替换，然后递归删掉那个替换节点。

## 6. 优先队列 / 二叉堆 (Priority Queue / Heap)

> PPT: DS06_Ch05_Priority Queues

### 题目

实现最小堆的插入和删除最小值（ Percolate Up / Down ）。

### 模板代码

```cpp
#define MIN_DATA -999999
#define MAX_SIZE 1000

typedef struct {
    int elements[MAX_SIZE];
    int size;
} PriorityQueue;

PriorityQueue* init(int maxSize) {
    PriorityQueue *H = (PriorityQueue*)malloc(sizeof(PriorityQueue));
    H->size = 0;
    H->elements[0] = MIN_DATA;  // 哨兵
    return H;
}

// 插入：Percolate Up（上滤）
void insert(PriorityQueue *H, int x) {
    int i;
    for (i = ++H->size; H->elements[i / 2] > x; i /= 2)
        H->elements[i] = H->elements[i / 2];
    H->elements[i] = x;
}

// 删除最小值：Percolate Down（下滤）
int deleteMin(PriorityQueue *H) {
    int min = H->elements[1];
    int last = H->elements[H->size--];
    int i, child;

    for (i = 1; i * 2 <= H->size; i = child) {
        child = i * 2;
        // 找较小的孩子
        if (child != H->size &&
            H->elements[child] > H->elements[child + 1])
            child++;
        // 如果 last 比孩子大，孩子上移
        if (last > H->elements[child])
            H->elements[i] = H->elements[child];
        else
            break;
    }
    H->elements[i] = last;
    return min;
}
```

| 操作 | 复杂度 |
|------|--------|
| 插入 `insert` | $O(\log n)$ |
| 删除最小 `deleteMin` | $O(\log n)$ |
| 建堆 `buildHeap` | $O(n)$（批量下滤） |

> 🔑 **哨兵 `elements[0]`** 设为极小值，使上滤的终止条件统一。**注意** `child != H->size` 防止比较不存在的兄弟。

## 7. 并查集 (Union-Find / Disjoint Set)

> PPT: DS07_Ch08_Union and Find

### 题目

$N$ 个元素，支持 `Union`（合并）和 `Find`（查根），判断两元素是否在同一集合。

### 模板代码

```cpp
#include <stdio.h>
#define MAXN 10001

int parent[MAXN];
int sz[MAXN];  // 按大小合并 / 按秩合并

void init(int n) {
    for (int i = 1; i <= n; i++) {
        parent[i] = i;
        sz[i] = 1;
    }
}

// 路径压缩
int find(int x) {
    if (parent[x] != x)
        parent[x] = find(parent[x]);  // 递归压缩
    return parent[x];
}

// 按大小合并 (Union by Size)
void unionSet(int a, int b) {
    int ra = find(a), rb = find(b);
    if (ra == rb) return;

    if (sz[ra] < sz[rb]) {
        parent[ra] = rb;
        sz[rb] += sz[ra];
    } else {
        parent[rb] = ra;
        sz[ra] += sz[rb];
    }
}

// 连通分量计数
int countComponents(int n) {
    int cnt = 0;
    for (int i = 1; i <= n; i++)
        if (parent[i] == i) cnt++;
    return cnt;
}
```

| 操作 | 复杂度 |
|------|--------|
| `find` + `unionSet` | 均摊 $O(\alpha(N))$ ≈ 常数 |

> 🔑 **两个优化缺一不可**：路径压缩 + 按大小/秩合并 → 几乎常数时间。

## 8. 线段树 (Segment Tree)

> PPT: DS07_Segment Tree

### 题目

维护一个数组，支持**单点修改**和**区间求和查询**。

### 模板代码

```cpp
#include <stdio.h>
#define MAXN 100005

int tree[MAXN * 4];
int A[MAXN];

void build(int node, int start, int end) {
    if (start == end) {
        tree[node] = A[start];
        return;
    }
    int mid = (start + end) / 2;
    build(node * 2, start, mid);
    build(node * 2 + 1, mid + 1, end);
    tree[node] = tree[node * 2] + tree[node * 2 + 1];
}

void update(int node, int start, int end, int idx, int val) {
    if (start == end) {
        tree[node] = val;
        return;
    }
    int mid = (start + end) / 2;
    if (idx <= mid)
        update(node * 2, start, mid, idx, val);
    else
        update(node * 2 + 1, mid + 1, end, idx, val);
    tree[node] = tree[node * 2] + tree[node * 2 + 1];
}

int query(int node, int start, int end, int L, int R) {
    if (L > end || R < start) return 0;       // 完全不在范围内
    if (L <= start && end <= R) return tree[node]; // 完全在范围内
    int mid = (start + end) / 2;
    return query(node * 2, start, mid, L, R)
         + query(node * 2 + 1, mid + 1, end, L, R);
}
```

| 操作 | 复杂度 |
|------|--------|
| 建树 `build` | $O(n)$ |
| 单点修改 `update` | $O(\log n)$ |
| 区间查询 `query` | $O(\log n)$ |

> 🔑 **三个 if** 是线段树查询的精髓：完全不在 → 返回 0；完全包含 → 返回节点值；部分重叠 → 递归两边。

## 9. 图与拓扑排序 (Graph & Topological Sort)

> PPT: DS08_Ch09_Graph Definition_Topological Sort

### 题目

给定 $N$ 门课程和 $M$ 个先修关系（A → B 表示 A 必须先于 B），求一个合法的上课顺序。如果存在环则输出 `impossible`。

### 模板代码

```cpp
#include <stdio.h>
#define MAXN 1005

int indegree[MAXN];
int graph[MAXN][MAXN];  // graph[u][v] = 1 表示 u→v

void topoSort(int N) {
    int q[MAXN], front = 0, rear = 0;
    int result[MAXN], cnt = 0;

    // 所有入度为 0 的节点入队
    for (int i = 1; i <= N; i++)
        if (indegree[i] == 0)
            q[rear++] = i;

    while (front < rear) {
        int u = q[front++];
        result[cnt++] = u;

        for (int v = 1; v <= N; v++) {
            if (graph[u][v]) {
                indegree[v]--;
                if (indegree[v] == 0)
                    q[rear++] = v;
            }
        }
    }

    if (cnt < N)
        printf("impossible\n");  // 有环
    else
        for (int i = 0; i < cnt; i++)
            printf("%d ", result[i]);
}
```

| 操作 | 邻接矩阵 | 邻接表 |
|------|----------|--------|
| 时间 | $O(N^2)$ | $O(N+M)$ |
| 空间 | $O(N^2)$ | $O(N+M)$ |

> 🔑 **入度为零 → 入队 → 删边（邻居入度减一）**。最终若输出节点数 < N，说明有环。

## 10. 最短路径 (Dijkstra)

> PPT: DS09_Ch09_Shortest-Path Algorithm

### 题目

给定有向图，求源点 $S$ 到所有节点的最短距离。边权非负。

### 模板代码

```cpp
#include <stdio.h>
#define MAXN 1005
#define INF 0x3f3f3f3f

int dist[MAXN];
int visited[MAXN];
int graph[MAXN][MAXN];

void dijkstra(int N, int S) {
    // 初始化
    for (int i = 1; i <= N; i++) {
        dist[i] = INF;
        visited[i] = 0;
    }
    dist[S] = 0;

    for (int i = 1; i <= N; i++) {
        // 找未访问中 dist 最小的
        int u = -1, minDist = INF;
        for (int j = 1; j <= N; j++) {
            if (!visited[j] && dist[j] < minDist) {
                minDist = dist[j];
                u = j;
            }
        }
        if (u == -1) break;  // 不可达节点

        visited[u] = 1;

        // 松弛操作
        for (int v = 1; v <= N; v++) {
            if (!visited[v] && graph[u][v] < INF) {
                if (dist[u] + graph[u][v] < dist[v])
                    dist[v] = dist[u] + graph[u][v];
            }
        }
    }
}
```

| 实现方式 | 复杂度 |
|----------|--------|
| 朴素版（邻接矩阵） | $O(V^2)$ |
| 堆优化（邻接表 + priority_queue） | $O((V+E)\log V)$ |

> 🔑 **松弛 (Relaxation)**：`dist[v] = min(dist[v], dist[u] + w(u,v))` 是 Dijkstra 的核心操作。

## 11. 排序算法汇总

> PPT: DS11~DS13 (Insertion, Shell, Heap, Merge, Quick, Bucket Sort)

### 快速排序 (QuickSort) — 最重要

```cpp
void quickSort(int arr[], int left, int right) {
    if (left >= right) return;

    // 三数取中选 pivot
    int mid = (left + right) / 2;
    if (arr[left] > arr[mid]) { int t = arr[left]; arr[left] = arr[mid]; arr[mid] = t; }
    if (arr[left] > arr[right]) { int t = arr[left]; arr[left] = arr[right]; arr[right] = t; }
    if (arr[mid] > arr[right]) { int t = arr[mid]; arr[mid] = arr[right]; arr[right] = t; }
    int pivot = arr[mid];
    // 将 pivot 换到 right-1
    { int t = arr[mid]; arr[mid] = arr[right - 1]; arr[right - 1] = t; }

    int i = left, j = right - 1;
    while (1) {
        while (arr[++i] < pivot);
        while (arr[--j] > pivot);
        if (i < j) { int t = arr[i]; arr[i] = arr[j]; arr[j] = t; }
        else break;
    }
    // 将 pivot 放回正确位置
    { int t = arr[i]; arr[i] = arr[right - 1]; arr[right - 1] = t; }

    quickSort(arr, left, i - 1);
    quickSort(arr, i + 1, right);
}
```

### 归并排序 (MergeSort)

```cpp
void merge(int arr[], int tmp[], int left, int mid, int right) {
    int i = left, j = mid + 1, k = left;
    while (i <= mid && j <= right)
        tmp[k++] = (arr[i] <= arr[j]) ? arr[i++] : arr[j++];
    while (i <= mid)  tmp[k++] = arr[i++];
    while (j <= right) tmp[k++] = arr[j++];
    for (i = left; i <= right; i++) arr[i] = tmp[i];
}

void mergeSort(int arr[], int tmp[], int left, int right) {
    if (left >= right) return;
    int mid = (left + right) / 2;
    mergeSort(arr, tmp, left, mid);
    mergeSort(arr, tmp, mid + 1, right);
    merge(arr, tmp, left, mid, right);
}
```

### 堆排序 (HeapSort)

```cpp
// 下滤：将以 root 为根的子树调整为最大堆
void percDown(int arr[], int root, int n) {
    int parent = root, child;
    int x = arr[root];
    for (; parent * 2 + 1 < n; parent = child) {
        child = parent * 2 + 1;
        if (child + 1 < n && arr[child] < arr[child + 1])
            child++;
        if (x >= arr[child]) break;
        arr[parent] = arr[child];
    }
    arr[parent] = x;
}

void heapSort(int arr[], int n) {
    // 建堆 O(n)
    for (int i = n / 2 - 1; i >= 0; i--)
        percDown(arr, i, n);
    // 排序 O(n log n)
    for (int i = n - 1; i > 0; i--) {
        int t = arr[0]; arr[0] = arr[i]; arr[i] = t;  // 交换堆顶和末尾
        percDown(arr, 0, i);  // 调整 heap size = i
    }
}
```

### 各排序对比

| 排序算法 | 平均 | 最坏 | 最好 | 空间 | 稳定 |
|----------|------|------|------|------|------|
| 插入排序 | $O(n^2)$ | $O(n^2)$ | $O(n)$ | $O(1)$ | ✅ |
| 希尔排序 | $O(n^{1.3})$ | $O(n^2)$ | $O(n)$ | $O(1)$ | ❌ |
| 堆排序 | $O(n\log n)$ | $O(n\log n)$ | $O(n\log n)$ | $O(1)$ | ❌ |
| 归并排序 | $O(n\log n)$ | $O(n\log n)$ | $O(n\log n)$ | $O(n)$ | ✅ |
| 快速排序 | $O(n\log n)$ | $O(n^2)$ | $O(n\log n)$ | $O(\log n)$ | ❌ |

> 🔑 **快排最坏情况**：pivot 选得不好（已排序数组取第一个 → $O(n^2)$）。**三数取中**或**随机 pivot** 可以避免。

## 12. 哈希表 (Hashing)

> PPT: DS14_Ch07_Hashing

### 题目

实现哈希表，用**平方探测** (Quadratic Probing) 解决冲突，支持插入和查找。

### 模板代码

```cpp
#include <stdio.h>
#include <stdlib.h>
#define EMPTY -999999

typedef struct {
    int *table;
    int size;  // 表大小，建议取素数
} HashTable;

HashTable* createTable(int size) {
    HashTable *H = (HashTable*)malloc(sizeof(HashTable));
    H->size = size;
    H->table = (int*)malloc(sizeof(int) * size);
    for (int i = 0; i < size; i++)
        H->table[i] = EMPTY;
    return H;
}

int hash(int key, int size) {
    return key % size;
}

// 平方探测插入
void insert(HashTable *H, int key) {
    int pos = hash(key, H->size);
    for (int i = 0; i < H->size; i++) {
        int probe = (pos + i * i) % H->size;  // f(i) = i²
        if (H->table[probe] == EMPTY) {
            H->table[probe] = key;
            return;
        }
    }
    printf("Insert failed: table full\n");
}

// 平方探测查找
int find(HashTable *H, int key) {
    int pos = hash(key, H->size);
    for (int i = 0; i < H->size; i++) {
        int probe = (pos + i * i) % H->size;
        if (H->table[probe] == EMPTY)
            return -1;  // 遇到空就说明不存在
        if (H->table[probe] == key)
            return probe;
    }
    return -1;
}
```

| 冲突解决 | 探测序列 $f(i)$ | 注意 |
|----------|----------------|------|
| 线性探测 | $i$ | 一次聚集 |
| 平方探测 | $i^2$ | 二次聚集；表大小应为素数 |
| 双散列 | $i \cdot h_2(key)$ | 几乎无聚集 |

> 🔑 **平方探测**：表大小为素数且**装填因子 < 0.5** 时一定能找到空位。遇到 `EMPTY` 即可终止查找。
