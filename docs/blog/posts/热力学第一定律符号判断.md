# 大学物理热力学第一定律符号判断

> 适用教材：《大学物理》（张三慧/马文蔚/程守洙）等  
> 记录时间：2026/06/18

---

## 1. 热力学第一定律的两种写法

热力学第一定律本质是**能量守恒定律**在热力学过程中的表达，但在不同教材中有两种写法：

| 写法 | 公式 | W 的含义 | 常见于 |
|------|------|----------|--------|
| **中国主流** | $\Delta U = Q + W$ | 外界对系统**做的功** | 国内教材（张三慧、马文蔚等） |
| **国际惯例** | $\Delta U = Q - W$ | 系统对外界**做的功** | 国外教材、物理化学课 |

> ⚠️ **关键区分**：两道公式里的 W **是相反的物理量**。
> - 中国版 W = **外界对系统做功**（压缩→正，膨胀→负）
> - 国际版 W = **系统对外界做功**（膨胀→正，压缩→负）

**本文统一采用中国教材的写法：$\Delta U = Q + W$**

---

## 2. 核心规则：符号判断一句话总结

> 🔑 **Q：吸热为正，放热为负**  
> 🔑 **W：外界对系统做功为正，系统对外界做功为负**

---

## 3. Q（热量）的符号判断

### 判断原则
把**系统**当主角，看热量的流动方向：

| 情况 | 符号 | 物理意义 |
|------|------|----------|
| 系统**从外界吸热** | **Q > 0**（正） | 热量流入系统，内能增加 |
| 系统**向外界放热** | **Q < 0**（负） | 热量流出系统，内能减少 |
| **绝热过程** | **Q = 0** | 无热交换 |

### 实际判断方法

1. **看温度变化方向**（同种物质，无相变）：
   - 系统温度升高 → 内能增大 → **通常是吸热** Q > 0
   - 系统温度降低 → 内能减小 → **通常是放热** Q < 0

2. **看相变方向**：
   - 熔化、汽化 → **吸热** Q > 0
   - 凝固、液化 → **放热** Q < 0

3. **用热容量计算**：
   $$Q = n C_m \Delta T \quad \text{或} \quad Q = mc \Delta T$$
   - $\Delta T > 0$ → Q > 0
   - $\Delta T < 0$ → Q < 0

---

## 4. W（功）的符号判断 🔥

> 这是最容易出错的地方。

### 判断原则

| 情况 | 符号 | 通俗理解 |
|------|------|----------|
| 外界**压缩**系统（体积减小） | **W > 0**（正） | 外界往里推 → 外界对系统做正功 |
| 系统**膨胀**（体积增大） | **W < 0**（负） | 系统往外推 → 系统对外界做正功 = 外界对系统做负功 |
| **等容过程** | **W = 0** | 体积不变，无功的交换 |

### 用公式判断

对于准静态过程（气体），外界对系统做功：

$$W = -\int_{V_1}^{V_2} p \, dV$$

> 💡 **注意**：这里的 p 是**系统的压强**。如果过程中压强不是常数，需要用积分。

- $V_2 > V_1$（膨胀）→ 积分值为正 → W 为**负**（系统对外做功）
- $V_2 < V_1$（压缩）→ 积分值为正 → W 为**负**

实际上更直观的理解：

$$W = \int_{V_1}^{V_2} (-p) \, dV$$

- 膨胀 $\Delta V > 0$：$-p \cdot \Delta V < 0$ → **W < 0**
- 压缩 $\Delta V < 0$：$-p \cdot (-\Delta V) > 0$ → **W > 0**

### 常见过程的 W 计算

| 过程 | 特征 | W 的表达式 |
|------|------|-----------|
| **等容** (isochoric) | $V = \text{const}$ | $W = 0$ |
| **等压** (isobaric) | $p = \text{const}$ | $W = -p(V_2 - V_1)$ |
| **等温** (isothermal) | $T = \text{const}$ | $W = -nRT \ln\frac{V_2}{V_1}$ |
| **绝热** (adiabatic) | $Q = 0$ | $W = \frac{p_1V_1 - p_2V_2}{\gamma - 1}$ |
| **多方** (polytropic) | $pV^n = \text{const}$ | $W = \frac{p_1V_1 - p_2V_2}{n - 1}$ |

## 5. 综合判断流程 🗺️

遇到热力学第一定律的题目时，按以下步骤走：

```
1. 明确系统是什么（气体？液体？）
        ↓
2. 判断过程类型（等温/等压/等容/绝热？）
        ↓
3. 判断 Q 的符号：
   - 吸热 → Q > 0
   - 放热 → Q < 0
   - 绝热 → Q = 0
        ↓
4. 判断 W 的符号：
   - 压缩（ΔV < 0）→ W > 0
   - 膨胀（ΔV > 0）→ W < 0
   - 等容（ΔV = 0）→ W = 0
        ↓
5. 代入 ΔU = Q + W 计算
        ↓
6. 检查 ΔU 的符号是否合理：
   - 温度升高 → ΔU > 0 ✓
   - 温度降低 → ΔU < 0 ✓
```

## 6. 典型例题

### 例题 1：等压膨胀

> 1 mol 理想气体在等压条件下从温度 $T_1$ 膨胀到 $T_2$（$T_2 > T_1$），求 Q、W、ΔU。

**分析**：
1. 等压过程，$p = \text{const}$
2. $T_2 > T_1$，温度升高 → **Q > 0**（系统吸热）→ **ΔU > 0**（内能增加）
3. 膨胀 → **W < 0**（系统对外做功）

**计算**：
- $W = -p(V_2 - V_1) = -R(T_2 - T_1)$  ← 注意是**负值**
- $Q = nC_p(T_2 - T_1) = C_p(T_2 - T_1)$  ← **正值**
- $\Delta U = Q + W = C_p(T_2 - T_1) - R(T_2 - T_1) = C_V(T_2 - T_1)$  ← **正值**

> ✅ 验证：温度升高 → ΔU > 0 ✓（因为 $C_V(T_2 - T_1) > 0$）

### 例题 2：等温压缩

> 1 mol 理想气体在等温条件下体积压缩为原来的一半，求 Q、W、ΔU。

**分析**：
1. 等温过程，$T = \text{const}$
2. 温度不变 → **$\Delta U = 0$**（理想气体内能只与温度有关）
3. 压缩 → **W > 0**（外界对系统做正功）
4. 由 $\Delta U = Q + W = 0$ 得 $Q = -W$ → **Q < 0**（系统放热）

**计算**：
- $\Delta U = 0$
- $W = -nRT \ln\frac{V_2}{V_1} = -RT \ln\frac{1}{2} = RT \ln 2$  ← **正值**（压缩 $V_2 < V_1$，$\ln < 0$，前面有负号 → W > 0）
- $Q = -W = -RT \ln 2$  ← **负值**（放热）

> ✅ 验证：压缩时外界对系统做功 → 内能本应增加 → 但等温下放热抵消 → ΔU = 0 ✓

### 例题 3：绝热膨胀

> 理想气体绝热膨胀，求 Q、W、ΔU 的符号。

**分析**：
1. 绝热过程 → **Q = 0**
2. 膨胀 → **W < 0**（系统对外做功）
3. $\Delta U = Q + W = 0 + W < 0$ → **ΔU < 0**（内能减少）

> 物理图像：绝热膨胀时，系统「自掏腰包」对外做功，内能降低，温度下降。

## 7. 常见陷阱 ❌

### 陷阱 1：混淆两种 W 的约定

| 中国教材 | 国际教材 |
|----------|----------|
| $\Delta U = Q + W_{\text{外对系}}$ | $\Delta U = Q - W_{\text{系对外}}$ |

一条题给「系统对外做功 100J」，则：
- 中国版：$W = -100\text{J}$（外界对系统做功为 −100J）
- 国际版：$W = 100\text{J}$（系统对外功为 100J）

> ⚠️ **考试前先确认老师用哪种约定！**

### 陷阱 2：惯性认为「做功就是正功」

只看「有没有做功」不看「谁对谁做功」是错误的。必须区分：
- **外界对系统做功** → W > 0（压缩过程）
- **系统对外界做功** → W < 0（膨胀过程）

### 陷阱 3：忘记等容过程 W = 0

等容（体积不变）时，无论压强如何变化，**气体体积不变则没有宏观位移**，W = 0。

### 陷阱 4：等温过程 ΔU ≠ 0（对非理想气体）

对于理想气体，等温 ΔU = 0。但对于实际气体或液体，等温过程内能也可能改变。

## 8. 速查表

| 过程 | ΔV | Q 符号 | W 符号 | ΔU 符号 |
|------|-----|--------|--------|---------|
| 等压膨胀 | > 0 | +（吸热） | −（系对外做功） | 取决于 Q 和 W 的大小 |
| 等压压缩 | < 0 | −（放热） | +（外对系做功） | 取决于 Q 和 W 的大小 |
| 等温膨胀 | > 0 | +（吸热） | −（系对外做功） | **0**（理想气体） |
| 等温压缩 | < 0 | −（放热） | +（外对系做功） | **0**（理想气体） |
| 等容升温 | = 0 | +（吸热） | **0** | +（内能增加） |
| 等容降温 | = 0 | −（放热） | **0** | −（内能减少） |
| 绝热膨胀 | > 0 | **0** | −（系对外做功） | −（内能减少） |
| 绝热压缩 | < 0 | **0** | +（外对系做功） | +（内能增加） |

## 9. 记忆口诀

::: tip 🎯 口诀
**「吸正放负，压正胀负」**

- 吸（吸热）→ Q 正
- 放（放热）→ Q 负
- 压（压缩）→ W 正
- 胀（膨胀）→ W 负
:::
