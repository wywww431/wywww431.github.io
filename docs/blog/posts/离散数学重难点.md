## 2.3 函数 (Functions)

### 📌 核心概念

#### 1. 函数的定义
- 设 A、B 为非空集合，函数 $f: A \to B$ 是 A 到 B 的映射
- **定义域 (Domain)**：A
- **陪域 (Codomain)**：B
- **值域 (Range)**：$f(A) = \{f(a) \mid a \in A\}$，值域 ⊆ 陪域
- 每个 A 中的元素**恰好对应一个** B 中的元素

#### 2. 函数的三种特殊类型 ⭐⭐⭐

| 类型 | 英文名 | 条件 | 记忆要点 |
|------|--------|------|----------|
| **单射** | Injective (One-to-One) | $a \neq b \Rightarrow f(a) \neq f(b)$ | 不同输入 → 不同输出 |
| **满射** | Surjective (Onto) | $\forall b \in B, \exists a \in A, f(a)=b$ | 陪域中每个元素都有原像 |
| **双射** | Bijective (One-to-One Correspondence) | 同时是单射和满射 | 一对一的完美匹配 |

> 🔑 **判断方法**：
> - $f(x)=x^2$（整数→整数）：**不是**单射（$f(-2)=f(2)=4$），**不是**满射（无整数平方=3）
> - $f(x)=x+1$（整数→整数）：**是**单射，**是**满射，**是**双射

#### 3. 反函数 (Inverse Function) ⭐⭐
- **仅当 f 是双射时才存在反函数** $f^{-1}$
- $f^{-1}(y)=x \iff f(x)=y$
- **易错点**：必须先验证 f 是否为双射，否则反函数不存在

#### 4. 函数复合 (Composition) ⭐
- $(f \circ g)(a) = f(g(a))$
- **前提条件**：g 的值域必须是 f 定义域的子集，否则复合无意义
- **注意顺序**：$f \circ g$ 是先算 g 再算 f
- 一般 $f \circ g \neq g \circ f$（不可交换）

#### 5. 取整函数 (Floor & Ceiling) ⭐⭐

| 函数 | 记号 | 含义 | 示例 |
|------|------|------|------|
| 下取整 floor | $\lfloor x \rfloor$ | 不大于 x 的最大整数 | $\lfloor 1.5\rfloor=1$, $\lfloor -0.5\rfloor=-1$ |
| 上取整 ceiling | $\lceil x \rceil$ | 不小于 x 的最小整数 | $\lceil 1.5\rceil=2$, $\lceil -0.5\rceil=0$ |

**关键性质**：
- $\lfloor -x \rfloor = -\lceil x \rceil$，$\lceil -x \rceil = -\lfloor x \rfloor$
- $\lfloor x+m \rfloor = \lfloor x \rfloor + m$（m 为整数）
- $x-1 < \lfloor x \rfloor \leq x \leq \lceil x \rceil < x+1$

### ❌ 常见错误
1. 混淆 **codomain（陪域）** 和 **range（值域）**——值域是陪域的子集
2. 判断 onto 时，只看「有没有元素映射过去」，忘了看「陪域中**所有**元素是否都有原像」
3. 求反函数前忘记检查**双射条件**
4. Floor 的负数情况：$\lfloor -1.5 \rfloor = -2$，不是 -1

## 2.4 序列与求和 (Sequences & Summations)

### 📌 核心概念

#### 1. 序列的定义
- 序列是从整数子集到集合 S 的**函数**
- 记作 $\{a_n\}$，$a_n$ 称为序列的项
- **顺序很重要**，不同于集合

#### 2. 两种重要序列 ⭐⭐

| 类型 | 形式 | 参数 |
|------|------|------|
| **等比数列 (Geometric)** | $a, ar, ar^2, \dots, ar^n, \dots$ | 初项 a，公比 r |
| **等差数列 (Arithmetic)** | $a, a+d, a+2d, \dots, a+nd, \dots$ | 初项 a，公差 d |

#### 3. 找序列规律的方法 ⭐
给定前几项，推断通项公式时考虑：
1. 相邻项是否有**相同差值**？（→ 等差数列）
2. 相邻项是否有**相同比值**？（→ 等比数列）
3. 是否由前项通过某种**组合**得到？（→ 递推）
4. 是否出现**循环**？
5. 是否存在连续相同值的**块**？

#### 4. 求和记号
$$\sum_{i=m}^{n} a_i = a_m + a_{m+1} + \cdots + a_n$$

- i：求和下标 (index)
- m：下限 (lower limit)
- n：上限 (upper limit)

## 2.5 集合的基数 (Cardinality of Sets) 🔥🔥🔥

> 这是整个第 2 章的**最大难点**，涉及无穷集合的「大小」比较。

### 📌 核心概念

#### 1. 基数的定义 ⭐⭐
- $|A| = |B|$ 当且仅当存在从 A 到 B 的**双射 (bijection)**
- $|A| \leq |B|$ 当且仅当存在从 A 到 B 的**单射 (injection)**
- $|A| < |B|$ 当且仅当 $|A| \leq |B|$ 且 $|A| \neq |B|$

#### 2. 可数与不可数 ⭐⭐⭐

| 术语 | 英文 | 含义 |
|------|------|------|
| **可数** | Countable | 有限集，或与 Z⁺ 有相同基数 |
| **可数无穷** | Countably Infinite | $|A| = |Z^+| = \aleph_0$ |
| **不可数** | Uncountable | 不是可数的集合 |

#### 3. 可数无穷集的例子 ⭐⭐
- **正整数集** Z⁺（显然）
- **正偶数集** E：$f(x)=2x$ 是 Z⁺ → E 的双射 → $|E| = |Z^+|$  
  → 说明**真子集可以与原集合有相同的基数**（无穷集特有！）
- **整数集** Z：可将 Z 排成序列 0, 1, -1, 2, -2, ... → 可数
- **正有理数集** Q⁺：用对角线枚举法 → 可数
- **全体有理数集** Q：可数

#### 4. Cantor 对角线论证法 ⭐⭐⭐ 🔥
> **本章最重要的证明技巧**

**定理**：(0,1) 区间上的实数集是**不可数**的。

**证明思路**：
1. 假设 (0,1) 可数，列出所有实数：$r_1, r_2, r_3, \dots$
2. 用小数展开表示：$r_i = 0.d_{i1}d_{i2}d_{i3}\dots$
3. 构造新数 $x = 0.x_1x_2x_3\dots$，其中：
   - 若 $d_{ii} \neq 3$，则 $x_i = 3$
   - 若 $d_{ii} = 3$，则 $x_i = 4$
4. $x$ 与列表中**每个数都至少有一位不同** → $x$ 不在列表中
5. 矛盾 → (0,1) 不可数

> 💡 **核心技巧**：通过改变对角线上的数字构造一个「不在名单上」的元素。

#### 5. Schröder-Bernstein 定理 ⭐
- 若 $|A| \leq |B|$ 且 $|B| \leq |A|$，则 $|A| = |B|$
- 等价表述：若存在 A→B 和 B→A 的单射，则存在 A→B 的双射

#### 6. 重要结论汇总

| 结论 | 说明 |
|------|------|
| $|(0,1)|$ 不可数 | Cantor 对角线法证明 |
| $|\mathbb{R}| = \mathfrak{c}$（连续统） | 用 tan(x) 建立 (0,1) 到 R 的双射 |
| 幂集基数 > 原集基数 | $\forall A, |A| < |\mathcal{P}(A)|$ |
| 连续统假设 (CH) | 不存在基数介于 $\aleph_0$ 和 $\mathfrak{c}$ 之间的集合 |

### ❌ 常见错误
1. **混淆可数与有限**：可数包含有限 + 可数无穷
2. **想当然认为无穷集都一样大**：正偶数集=正整数集 ✓，但实数集更多 ✗
3. **对角线法误用**（见 PPT slide 58-59 的错误证明示例）
4. **认为 Q 和 R 一样大**：Q 是可数的，R 是不可数的

## 📝 典型题型

1. **判断函数类型**：给定 f 的定义，判断单射/满射/双射
2. **求反函数**：先验证双射，再求 $f^{-1}$
3. **计算复合函数**：$f \circ g$
4. **Floor/Ceiling 等式证明**：设 $x = n + \varepsilon$，分情况讨论
5. **找序列通项**：给定前几项，推测公式
6. **证明两集合基数相等**：构造双射
7. **用对角线法证明不可数**：熟练默写证明步骤
8. **判断集合是否可数**：Z⁺、E、Z、Q 可数；(0,1)、R 不可数
